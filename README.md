# CRM / Client Management Dashboard UI

## 📌 Project Overview
This project is a CRM (Client Relationship Management) Dashboard UI designed for small agencies and service-based businesses. The dashboard helps users manage clients, track leads, monitor tasks, and improve workflow efficiency.

## 🎯 Objective
To create a clean and user-friendly dashboard interface that allows businesses to manage client information and daily activities in one place.

## ✨ Features
- Dashboard Overview
- Client Management
- Lead Tracking
- Task Tracking
- Follow-up Management
- Revenue Monitoring
- Activity Tracking

## 🖥️ Screens Included
### 1. Dashboard Home
- Total Clients
- Active Leads
- Revenue Overview
- Recent Activities

### 2. Client Management
- Client List
- Client Status Tracking
- Search and Filter Options

### 3. Task Tracker
- Daily Tasks
- Upcoming Tasks
- Follow-up Reminders
